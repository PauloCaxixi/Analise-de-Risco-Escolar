# Datathon – Passos Mágicos | Risco de Defasagem Escolar

## Objetivo
Construir um modelo de Machine Learning capaz de estimar o risco de defasagem
escolar de alunos da Associação Passos Mágicos.

## Solução
Pipeline completa de ML com:
- pré-processamento
- engenharia de atributos
- treinamento
- deploy via API FastAPI
- monitoramento de drift

## Stack
- Python 3.10+
- scikit-learn
- FastAPI
- Docker
- pytest
- Streamlit
